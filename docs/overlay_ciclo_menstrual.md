## 3) Como incorporar el ciclo al entrenamiento (guia operativa)

### 3.1 Fuerza

| Bloque                              | Estrategia por fases                                                                                                                                                                                                                                                                             |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Fuerza maxima (1-5 reps, % alto)    | Menstruacion temprana y luteal tardia: bajar a 70-85% 1RM (RPE 6-8), mas descanso; priorizar dobles/triples limpias. Folicular y ovulatoria: subir a 80-92% si la tecnica se mantiene; usar singles controladas; VBT con cut-off de velocidad (-20%). Evitar grinders y picos si sintomas altos. |
| Hipertrofia (6-15 reps)             | Menstruacion temprana: reducir 1 serie por grupo y RIR 2-3; variantes de menor carga axial. Folicular y ovulatoria: volumen objetivo; RIR 1-2; superseries antagonistas. Luteal tardia: bajar densidad (mas descanso) y usar tempo/pausas para mantener estimulo sin subir carga.                |
| Levantamientos olimpicos (potencia) | Menstruacion temprana: tecnica desde posiciones y complejos ligeros (60-75%). Folicular y ovulatoria: enfasis en potencia y singles tecnicos; pliometria moderada. Luteal tardia: evitar COD/saltos agresivos si laxitud/fatiga; tirones, posiciones, jerk supports.                             |
| Accesorios/estabilidad              | En dias con sintomas: core anti-extension/anti-rotacion, gluteo medio, rotadores externos; carries e isometrias.                                                                                                                                                                                 |

### 3.2 Resistencia, HIIT/MetCon y movilidad

| Bloque                               | Estrategia por fases                                                                                                                                                                                                                                        |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Resistencia (carrera/beep/bici/remo) | Menstruacion temprana: Z1-Z2 o tecnica; si entrenas calidad, intervalos menos densos. Folicular y ovulatoria: sesiones clave (VO2, umbral); progresion de ritmo. Luteal tardia: mantener volumen pero reducir picos; usar ergometros si el impacto molesta. |
| HIIT/MetCon                          | Menstruacion y luteal tardia: densidad moderada (trabajo:descanso >= 1:1), menos complejidad tecnica y menos impacto; priorizar sled/ergometros. Folicular y ovulatoria: aumentar intensidad o complejidad si la recuperacion es buena.                     |
| Pliometria/COD                       | Menstruacion temprana y luteal tardia: bajar altura y numero de contactos; progresion lateral suave. Folicular y ovulatoria: contactos moderados/altos segun historial; monitorizar aterrizajes.                                                            |
| Movilidad/recuperacion               | Aumentar dosis en menstruacion y luteal tardia; tobillo/cadera/toracica y respiracion 360 grados.                                                                                                                                                           |

## 6) Algoritmo de ajuste (pseudocodigo)

```text
1) Determina fase (dia del ciclo). Si anticoncepcion combinada -> fase plana.
2) Obten multiplicadores de fase y de sintomas (check-in 0-3).
3) intensity = base_intensity * clamp(M_fase_int * M_sint_int, 0.80, 1.10).
4) volume = base_volume * clamp(M_fase_vol * M_sint_vol, 0.80, 1.10).
5) rest = base_rest_sec + extra_fase + extra_sintomas.
6) Si impact_level >= 1 y (fase en {menstruacion temprana, luteal tardia} y sintomas >= 2) -> aplicar SWAP.
7) Si sintomas >= 2 durante 2 semanas -> deload (-20%) la siguiente semana.
8) Registrar RPE/cumplimiento para autoajuste +/- 2-5%.
```

## 7) Checklist rapido para que tu agente contraste contra el backend

- Determinacion de fase:
  - Mapeo dia 1-28 a una de las 7 fases.
  - Caso especial: anticoncepcion combinada -> fase plana (sin variaciones por fase).
- Modelo de modificadores:
  - Multiplicadores de fase para intensidad y volumen.
  - Multiplicadores por severidad de sintomas 0-3 para intensidad y volumen.
  - Extra de descanso por fase y por sintomas.
  - Clamp final de intensidad y volumen a [0.80, 1.10].
- Motor de sustituciones (swaps):
  - Condicion: impact_level >= 1 y fase (menstruacion temprana o luteal tardia) y sintomas >= 2.
  - Sustituciones por patron (plyo, COD, axial, overhead, core dinamico, calor/estres).
  - Campos de swaps en BD de ejercicios disponibles y resolubles por id.
- Deload por persistencia:
  - Regla: sintomas >= 2 durante 2 semanas -> siguiente semana -20% (definir donde se aplica: volumen, intensidad o ambos).
- Telemetria:
  - Registro de RPE y cumplimiento.
  - Autoajuste +/-2-5% con logica definida (por ejemplo por objetivo de RPE o adherencia).
