# Programacion completa: intermedio

Ruleset: `program-intermediate/2.0.0`. Objetivo: ampliar fuerza y potencia, consolidar pacing y combinar modalidades sin perder tecnica. High skills requieren permiso individual, no etiqueta global.

## Frecuencia y sesion

| Variable     | 3 dias minimo      | 4 dias recomendado        | Quinto dia                                |
| ------------ | ------------------ | ------------------------- | ----------------------------------------- |
| Duracion     | 55-65 min          | 55-70 min                 | solo zone 2, movilidad o skill sin fatiga |
| Distribucion | no D2 consecutivos | max secuencia D1-D2-D0/D1 | no aumenta cuota de D2                    |
| RPE >=8      | max 2/semana       | max 2/semana              | prohibido si fatiga alta                  |

Estructura: check-in 2, warm-up 8-12, skill/olimpico 10-15, fuerza 15-20, WOD 8-22, accesorio/cool-down 5-10. Fuerza y halterofilia pueden alternarse; no se apilan dos bloques pesados.

## Bloque de 10 semanas

| Semanas | Fase                  | Volumen       | Intensidad y control              |
| ------- | --------------------- | ------------- | --------------------------------- |
| 1       | baseline              | 70-75 %       | RPE 6, tests submaximos           |
| 2-4     | acumulacion           | 85/95/100 %   | RPE 6-8; una variable/semana      |
| 5       | descarga              | 60-70 % de W4 | sin benchmark/high skill fatigado |
| 6-8     | intensificacion       | 85/95/100 %   | carga sube, volumen no; max 2 D2  |
| 9       | realizacion           | 80-90 %       | una prueba comparable, resto D1   |
| 10      | descarga/reevaluacion | 55-65 %       | tests tecnicos y nivel            |

## Cuotas por semana de 4 dias

| Componente      | Cuota                                                             |
| --------------- | ----------------------------------------------------------------- |
| Fuerza          | 2-3 bloques; 4-6 x 2-6, RPE 6-8                                   |
| Halterofilia    | 2 exposiciones: tecnica + potencia; <=24 reps totales bajo fatiga |
| Gimnasia        | 2-3; al menos 1 estricta; kipping solo con permiso                |
| Mono            | 2; una base 20-35 min y una de intervalos/pacing                  |
| Metcon          | 3; corto, medio y variable en ventana de 14 dias                  |
| Largo 18-30 min | max 1/semana                                                      |
| Impacto alto    | max 2, separadas 36-48 h                                          |
| Bisagra pesada  | max 2, separadas >=48 h; solo una entra en metcon                 |
| Overhead denso  | max 2; no consecutivas                                            |
| Grip alto       | max 2; >=48 h entre trepa/C2B/TTB/KB denso                        |

Con 3 dias: fuerza 2, olimpico 1-2, gimnasia 2, mono 2 y metcon 2-3; se cubren cuotas en 14 dias. No se añade volumen por sesion para imitar cuatro dias.

## Ejemplos

### Tres dias

| Dia  | Bloque principal                   | WOD                                                | Stimulus          |
| ---- | ---------------------------------- | -------------------------------------------------- | ----------------- |
| 1 D2 | front squat 5x4                    | AMRAP 12 row + wall ball + strict pull             | sostenido mixto   |
| 2 D1 | hang power snatch tecnico + zone 2 | intervalos 5 x 3/2                                 | pacing aerobico   |
| 3 D2 | strict press + gym skill           | For Time cap 15 bike + DB snatch + burpee adaptado | potencia repetida |

### Cuatro dias

| Dia  | Bloque principal             | WOD                 | Restriccion                     |
| ---- | ---------------------------- | ------------------- | ------------------------------- |
| 1 D2 | back squat                   | couplet 8-12 min    | sin salto alto al dia siguiente |
| 2 D1 | gimnasia estricta            | mono 20-30 min      | grip bajo                       |
| 3 D2 | power clean tecnico/potencia | intervals 12-18 min | sin bisagra pesada previa 48 h  |
| 4 D1 | upper pull/push + carry      | chipper 15-22 min   | complejidad media, RPE <=7      |

## Cargas y densidad

- Metcon ciclico: 30-50 % de 1RM tecnico; reps bajas: 50-65 %; nunca >75 % bajo fatiga.
- Olympic por set <=5, cero series al fallo; si hay dos misses consecutivos, bajar 10 % o cambiar a derivado.
- Trabajo por minuto EMOM debe acabar antes de segundo 50; dos rondas >50 s fuerzan scaling.
- Drift de intervalos >12 % activa reduccion 10-20 % de trabajo.
- Progresion de fuerza `+2,5-5 %`, volumen `+1 serie` o densidad `+3-5 %`, una sola variable.

## Interferencia

- No deadlift pesado y swing/row sprint denso en 48 h.
- No C2B/TTB alto volumen tras rope climb o farmer carry alto.
- No overhead squat/snatch fatigado tras pressing pesado o HSPU.
- Si hay carrera de impacto alto, el siguiente dia de tren inferior es fuerza controlada sin pliometria.
- Benchmark sustituye al metcon intenso semanal; no se suma.

## Salida

La promocion exige confianza alta, dimensiones avanzadas y validacion tecnica. Quien tiene motor 3 pero skill 1 sigue en programacion intermedia con mono individualizado.
