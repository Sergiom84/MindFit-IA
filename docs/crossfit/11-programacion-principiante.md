# Programacion completa: principiante

Ruleset: `program-beginner/2.0.0`. Objetivo: competencia de patrones, base aerobica, tolerancia gradual y autonomia de escalado. No se programa kipping, inversion dinamica ni levantamiento olimpico desde suelo bajo fatiga.

## Frecuencia y sesion

| Variable       | 2 dias                            | 3 dias recomendado   | Limite                                     |
| -------------- | --------------------------------- | -------------------- | ------------------------------------------ |
| Validez        | minimo viable, progreso mas lento | metodologia completa | 4.º dia solo recuperacion tecnica/aerobica |
| Duracion       | 45-55 min                         | 45-60 min            | trabajo efectivo intenso <=15 min          |
| Dias           | 48-72 h entre sesiones            | p. ej. L-X-V         | nunca D2 consecutivos                      |
| Metcon RPE >=8 | maximo 1 cada 2 semanas           | maximo 1/semana      | no si readiness <3/5                       |

Estructura: check-in 2-3 min, warm-up 8-10, skill 10-15, fuerza/tecnica 10-15, WOD 6-15, accesorios/cool-down 5-8. Si la sesion dura <45 min se elimina accesorio antes de warm-up o skill.

## Bloque de 8 semanas

| Semana | Objetivo                | Volumen relativo | Intensidad           | Skills nuevos     |
| ------ | ----------------------- | ---------------- | -------------------- | ----------------- |
| 1      | baseline y calibracion  | 70 %             | RPE 5-6              | max 2 regresiones |
| 2      | acumulacion             | 80 %             | RPE 6                | max 1             |
| 3      | acumulacion             | 90 %             | RPE 6-7              | max 1             |
| 4      | descarga                | 60-65 % de W3    | RPE <=6              | 0                 |
| 5      | reconstruccion          | 85 %             | RPE 6                | max 1             |
| 6      | progresion              | 95 %             | RPE 6-7              | max 1             |
| 7      | consolidacion           | 100 %            | una exposicion RPE 8 | 0 bajo fatiga     |
| 8      | descarga y reevaluacion | 55-65 %          | tests submaximos     | 0                 |

Solo se progresa una variable por sesion: carga `+2,5-5 %`, reps totales `+5 %`, duracion `+1-2 min` o descanso `-5-10 s`. Si tecnica <2/3, RPE >8 o dolor >1, no se progresa.

## Cuotas por semana de 3 dias

| Componente                             | Cuota                                                                              |
| -------------------------------------- | ---------------------------------------------------------------------------------- |
| Squat/hinge/lunge/push/pull/carry/core | cada patron >=1; squat y hinge no dominan ambos el mismo dia                       |
| Fuerza                                 | 2 bloques; 3-5 series x 3-8, RPE 5-7                                               |
| Halterofilia                           | 1-2 bloques tecnicos con PVC/barra/DB; derivados desde hang, <=12 reps bajo fatiga |
| Gimnasia                               | 2 bloques estrictos/isometricos; cero kipping                                      |
| Mono                                   | 2 exposiciones; una continua 10-20 min o intervalos 1:1-1:2                        |
| Metcon                                 | 2; 6-15 min, baja complejidad                                                      |
| Impacto alto                           | max 1; 48 h antes/despues                                                          |
| Bisagra cargada                        | max 1 moderada; 48 h                                                               |
| Overhead cargado                       | max 1; sin fatiga tecnica                                                          |
| Largo >20 min                          | max 1 cada 2 semanas; predominantemente mono                                       |

Con 2 dias se combinan full-body + mono y se mantienen todos los patrones en una ventana de 14 dias. Se avisa que no permite la misma velocidad de aprendizaje; no se densifica para compensar.

## Microciclos ejemplo

### Dos dias

| Dia | Skill/fuerza                  | WOD                                        | Stimulus          |
| --- | ----------------------------- | ------------------------------------------ | ----------------- |
| A   | squat + empuje/remo estricto  | AMRAP 8-10 de step-up, ring row y bike     | continuo, RPE 6-7 |
| B   | hinge + carry + barra tecnica | intervalos 6 x 60/60 row y deadlift ligero | pacing y postura  |

### Tres dias

| Dia           | Skill/fuerza                   | WOD                                                      | Dominante        |
| ------------- | ------------------------------ | -------------------------------------------------------- | ---------------- |
| 1 D1          | goblet squat 4x6 + landing     | EMOM 10: bike / step-up                                  | squat + aerobic  |
| 2 D1          | ring row/push-up + carry       | AMRAP 9: row, push-up adaptada, carry                    | upper + continuo |
| 3 D2 opcional | hinge 4x5 + hang clean tecnico | For Time cap 10: row + KB deadlift/swing ruso autorizado | hinge + mixto    |

Los ejemplos no son plantillas fijas. El generador rota modalidad y patron respetando cuotas y ventanas.

## Permisos y escalado

- Carga metcon ciclica <=35 % de 1RM tecnico estimado; baja repeticion <=50 %.
- Serie planificada <=60 % de capacidad fresca demostrada.
- Saltos solo tras 10 aterrizajes silenciosos y estables; si no, step-up.
- Run se sustituye por bike/row si impacto no autorizado.
- Pull-up = ring row/band strict; TTB = dead bug/hanging knee raise.
- Olympic = posiciones/derivados; cero misses planificados.

## Criterios de recorte

1. Si warm-up RPE >=9 a dosis ligera, convertir en recuperacion y evaluar sintomas.
2. Si queda <70 % del tiempo planificado, conservar warm-up + skill + WOD reducido; no comprimir descansos.
3. Si dos intervalos caen >15 %, reducir reps 20 % o ampliar descanso.
4. Si tecnica pasa a 0/3, parar el movimiento y sustituir; no solo bajar carga.
5. Si dolor aumenta >=2 puntos, aplicar stop rule de seguridad.

## Salida del bloque

Promocion solo con los criterios de `level-model/2.0.0`; completar 8 semanas no basta. Un principiante puede permanecer varios bloques variando capacidades sin subir complejidad.
